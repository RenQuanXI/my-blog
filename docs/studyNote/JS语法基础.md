5. continue   跳出本次循环，继续下一循环，

   <!--吃包子 有五个包子正在吃呢，迟到了第二个，第三个突然掉地上了，脏了， 就不捡它了，跳过它直接吃第四个，那么这掉在地上的第三个包子，就像相当于是这个continue 跳过这个继续吃下一个-->

   ​	for ：当循环次数确定时，一般选择使用for循环

   局部作用域：在函数体内的执行环境。

   局部变量：函数定义的形参，以及函数体中用var关键字定义的变量。

   作用域链：当访问一个变量时，会先从本作用域中去找这个变量，若找不到怎会向上级作用域中去找，以此类推，就形成了一个作用域链

   ```js
   //全局中的特殊情况
   function fn(){
       //在此a是未使用var关键字声明的变量；
       //当代码执行到当前位置时a会变成全局变量；
       //但是要是后面不去调用这个函数的话这个函数体内的内容就不会执行，也就不会将其变成全局的变量了 
       a = 123；
   }
   fn();
   alert(a);
   ```
